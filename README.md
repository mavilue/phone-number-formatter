# Phone Number Formatter (Java)

Simple Java application for validating and formatting phone numbers.

---

## Features

- Accepts raw or partially formatted input  
- Supports:
  - Landline (8 or 10 digits)
  - Mobile (9 or 11 digits)  
- Automatically detects phone type  
- Removes non-numeric characters  
- Formats numbers into standard patterns  

---

## Tech

- Java  
- String manipulation (`substring`, `replaceAll`)  
- Regular Expressions (Regex)  
- Conditional logic  

---

## Run

```bash
javac Main.java
java Main
